Score predictor

Web app allowing Sale Sharks rugby fans to guess the score of a game. Correct scores were entered into a draw to win a TV from the club's principal sponsor AO.com.

About the game

The app was designed to help Sale Sharks improve engagement, improve the quality of the email database and provide visibility to the new headline sponsor. Over 2000 played the game before I left the club.

How it worked

Built using Node with an ExpressJS backend the app would receive responses from the embedded game. Answers were stored in a MongoDB database until the match started and the entries were locked. After the game the correct score was entered via the admin and any winners were contacted.
